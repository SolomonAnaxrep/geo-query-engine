# Geo Query Engine

A simple geographic query engine frontend that supports Zippopotomus and OpenStreetMap services.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3031`

## Features

- Service selection buttons for Zippopotomus and OpenStreetMap
- Clean, modern UI with gradient design
- TypeScript for type safety
- Vite for fast development and building
